# Teste Avaliativo

1. Instale o Node.js
```bash
npm install
```

2. Configure o arquivo `.env`
```bash
cp example.env .env
```

3. Execute o arquivo

Para gerar um arquivo xlsx
```bash
node "Nome Do Arquivo"
```

Para gerar um arquivo json
```bash
node "Nome Do Arquivo" -d
```

Para somente listar no console
```bash
node "Nome Do Arquivo" -l
```
